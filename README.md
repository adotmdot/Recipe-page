<h2>⚙️ Installation</h2>

<h3>Clone Repository</h3>

<pre>
git clone https://github.com/adotmdot/Recipe-page.git
</pre>

<h3>Backend Setup</h3>

<pre>
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
</pre>

<h3>Frontend Setup</h3>

<pre>
cd charlene-ai-recipesnpm
npm install
npm run dev
</pre>
